# taskwarrior

> An open source todo application for your terminal.

- Add a Task:

`task add {{task}}`

- List all tasks:

`task list`

- Mask a task as done:

`task {{task_id}} done`

- Add a project to a task:

`task {{task_id}} modify project:{{project_title}}`

- Display all available commands:

`task commands`
